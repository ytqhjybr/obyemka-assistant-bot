🔍 Обзор проекта
ИИ-ассистент для компании ОБЪЕМКА 3D — это интеллектуальный чат-бот, разработанный для автоматизации коммуникации с клиентами в сфере аддитивных технологий. Ассистент помогает пользователям получать экспертную информацию о 3D-печати, подбирать материалы и технологии, а также записываться на консультации с инженерами компании.

Приложение Б: Глоссарий терминов
Термин
Определение
RAG
Retrieval-Augmented Generation — генерация ответов с подкреплением из внешней базы знаний
Intent
Намерение пользователя, определяемое ИИ для выбора сценария диалога
Fallback
Резервный сценарий при невозможности обработать запрос автоматически
Слот (Calendar)
Временной интервал, доступный для записи на встречу
Эскалация
Передача диалога от ИИ к живому специалисту с сохранением контекста
